## 安装 ##
如果已经安装好了Python，建议直接通过pip来安装PsychoPy，不另安装Standalone版的PsychoPy
### Python标准安装（推荐） ###
    pip install psychopy
### Python纯安装（后续手动安装依赖包-不推荐） ###
    pip install psychopy --no-deps
若只考虑使用PsychoPy的Builder模式，可直接安装PsychoPy的exe
